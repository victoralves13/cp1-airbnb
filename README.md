# CP1 - Airbnb Clone
Este projeto é sobre o site do Airbnb, desenvolvido como atividade da disciplina de Front-End (CP1).  
O site foi construído utilizando HTML, CSS e JavaScript, seguindo as regras e os requisitos do professor.
